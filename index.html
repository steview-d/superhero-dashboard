Introduction

This page contains data originally from the SHDB for over 700 different super heroes.

The data visualizations that follow give a high-level overview of this dataset and clicking on the data allows you to filter the results to get down to the specifics.

Ever wondered what color hair the bad guys and girls usually have? Me neither, but if you want to know, now you can!

You can use the data below to create the super hero with the most common features and attributes to create the ultimate all round best hero you can or go the opposite way and create the most obscure hero to ever exist - or anything in between!

Gender

A lot of the entries in this data set are from heroes who first appeared as early as the 1930's, therefore it's not overly surprising to see men outnumber women by 2.5 to 1.
